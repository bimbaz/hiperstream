
# Desafio Hiperstream

Este projeto é um desafio para a posição de Desenvolvedor na empresa Hiperstream. O objetivo é criar uma aplicação em linha de comando que leia um arquivo de entrada simulando dados de faturas de clientes fictícios, aplicando regras específicas e gerando arquivos de saída em formato CSV, de acordo com as regras estabelecidas.

## Motivo da escolha da linguagem Python

A escolha da linguagem Python se deu pela minha familiaridade com a linguagem e pela sua facilidade de uso e legibilidade de código. Além disso, Python possui uma grande comunidade de desenvolvedores e uma vasta biblioteca de módulos que podem ser utilizados para facilitar o desenvolvimento da aplicação.

## Instruções de instalação

Para criar um ambiente virtual, execute o seguinte comando no terminal:

```bash
python -m venv venv
```

Para ativar o ambiente virtual no Windows, execute o seguinte comando no terminal:
    
```bash
venv\Scripts\activate
```

Para ativar o ambiente virtual no Linux, execute o seguinte comando no terminal:
    
```bash
source venv/bin/activate
```

Para instalar as dependências do projeto, execute o seguinte comando no terminal:

```bash
pip install -r requirements.txt
```

## Como utilizar

Para executar a aplicação, execute o seguinte comando no terminal:

```bash
python main.py
```
